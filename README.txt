Project Tile : Sentiment Analysis of Facial Expression.

How to run applciaiton?

1. Exceute the main python jupyter notebook. 
2. It will pop-up a graphical user interface. Click on Open Camera Button on Menu Bar. 
3. Capture the image using SpaceBar key. Image will be saved in your file.
4. Close the window of the applicaiton(GUI). Then return to the jupyter notebook console.
5. You will see the graph and the video will be played that will be the output of our application.
